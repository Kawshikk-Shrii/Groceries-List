       <li><b> Installation of React </b></li>
       <pre>*  To Install React, first needed install node.js, and then from store.chrome.web:- download "React Developer Tools". After downloading in command prompt type the commands 
              node -v (It can be typed in vs code terminal also!!)</pre>
       <pre>*   Afterwards intall the extensions in the vs code that is react/redux/... and also ES7...</pre>
       <pre>*   After that, type the command 
                     npx create-react-app [*folder*].
                This enables react files to open up in the folders.</pre>
       <pre>*   After that directly change directory to that folder by typing the command 
                     cd [*folder*]
                     and then 
                     npm start              #it runs the react app!!</pre>
       <p>This is how it is installed and enabled.</p>
